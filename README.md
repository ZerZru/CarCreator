<h2> Installing: </h2>
<hr>
1) Download files

```git
  pip install CarCreator
```
2) Import game to Python 3.6 Console

```python
  import run_cc_v001_en
  from run_cc_v001_en import game_start
```
3) Start game

```python
  game_start()
```

4) Playing
