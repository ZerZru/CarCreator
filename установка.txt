Установка игры:
1) Скачать все файлы из репозитория на GitHub
1.1) Или скачать их с сайта (пока что не доступно)
2) Перенести скачанные файлы в коренную папку интерпретатора Python
3) Запустить Python коносль
4) В Python консоли прописать import run_cc_v001_ru
4.1) После прописать from run_cc_v001_ru import game_start()
5) Прописать game_start()
6) Играть
